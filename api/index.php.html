Boa's Client API

Boa provides a client API to programmatically use a remote Boa infrastructure. This allows writing programs in other languages that submit a query written in the Boa language and retrieve results back, similar to how SQL queries may be embedded in languages.

The idea is to let Boa do most of the heavy lifting of finding, filtering, and aggregating the data you need while the more complex uses of that aggregated data remain in another language, such as Java.

Currently, the API only supports the Java, C#, JavaScript and Python languages. In the future we may add additional support for languages such as C/C++, etc based on user feedback.

For more information, please see the documentation.

Otherwise, please head over to the releases page to download a release and start using the client API!
