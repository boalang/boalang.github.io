Dataset Statistics

This page contains statistics on the various datasets available in Boa.

Note that the latest dataset has smaller versions available (to make algorithm design and testing faster) which were created by randomly selecting projects from the full dataset. These small and medium datasets should not be used in research results, as they are removed when new datasets are ready!
